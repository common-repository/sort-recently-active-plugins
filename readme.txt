Sorts plugins in Recently Active list by deactivation date.

== Description ==

Sorts plugins in Recently Active list by deactivation date.

== Installation ==

1. Upload `display-active-plugins-first` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
